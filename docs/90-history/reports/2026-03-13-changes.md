### 2. Build System Improvements

#### deps/wxWidgets/wxWidgets.cmake
- **Pinned wxWidgets to specific commit:** `a9d946902685b9946d8775f07d2a73a9b5bef394`
- Added comment explaining the pin (BambuLab's bbl_deps merge)
- Prevents upstream breakage from master branch changes

#### CMakeLists.txt
- **Added ccache integration:**
  - Automatic detection with `find_program(CCACHE_PROGRAM ccache)`
  - Sets `CMAKE_C_COMPILER_LAUNCHER` and `CMAKE_CXX_COMPILER_LAUNCHER`
  - Informative messages for both found and not found cases
- Reduces rebuild time from 30-60 minutes to ~5 minutes
