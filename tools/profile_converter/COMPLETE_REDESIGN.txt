Features:

  ✅ Interactive brand selection
  ✅ Interactive printer selection per brand
  ✅ Number ranges (1-5)
  ✅ Klipper-only option (K)
  ✅ Conversion summary before proceeding
  ✅ Real-time progress

Selection Flow:
  1. Select brands (numbers, ranges, A, K, or Q)
  2. For each brand: All (A), Select (S), or Skip (K)
  3. If Select: choose specific printers
  4. Review summary
  5. Confirm and convert

Examples:
  • Enter "K" → Select all Klipper brands
  • Enter "1,3,5" → Select brands 1, 3, and 5
  • Enter "1-5" → Select brands 1 through 5
  • Enter "A" → Select all brands
